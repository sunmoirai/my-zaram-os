# 자람이 (zaram-i)

아이의 월령에 맞춘 **발달 가이드(정적)** 와 **오늘의 성장 기록**을 입력할 수 있는 간단한 웹앱입니다.

> 현재 버전은 **Gemini API 없이** 동작하도록 구성되어 있습니다.
> 발달 가이드는 네트워크/키 없이 항상 표시되며, 대신 AI 기반의 세밀한 문장 생성/개인화는 제공되지 않습니다.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
